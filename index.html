What it handles

Headings, paragraphs, bold text, italics, links, lists, tables and common document structure. For table-heavy Word documents, try Confluence wiki markup: it writes headers as ||header|| and cells as |cell|.

This static utility runs entirely in your browser, making it suitable for GitHub Pages.
